# clubhub

Basic frontend setup for ClubHub web interface. Main page is stored in App.vue, ClubHub logo SVG is stored in icon.vue under the "components" folder. Use assets folder to store additional images or site resources.

Currently using Vuetify to import prebuilt site components and manage overall page structure. Access Vuetify documentation under the "components" section to import new items and find guides for manipulating existing ones. 

I am currently running this project inside a docker container with NPM installed, and I recommend that you do too for development consistency. Once the container is running, you can use VScode extensions to attach your IDE to the container for ease of use.




This template should help get you started developing with Vue 3 in Vite.

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
